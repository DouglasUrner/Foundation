# Typography

## [Font Awesome](https://fontawesome.com)

1. `yarn add @fortawesome/fontawesome-free`
1. Add the following line to `app/javascript/packs/application.js`:

```javascript
import "@fortawesome/fontawesome-free/css/all.css";
```
